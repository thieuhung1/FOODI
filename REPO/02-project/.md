FULLSTACK WEB ROADMAP (HTML + CSS + JS + NODEJS + MONGODB)

========================================
I. TỔNG QUAN
============

Fullstack Developer = Frontend + Backend + Database

* Frontend: Giao diện (HTML, CSS, JS)
* Backend: Xử lý logic (NodeJS)
* Database: Lưu dữ liệu (MongoDB)

========================================
II. FRONTEND (CLIENT SIDE)
==========================

1. HTML (Structure)

* Dùng để tạo khung web
* Các thẻ quan trọng:

  <div>, <header>, <footer>, <section>, <article>
  <form>, <input>, <button>

Ví dụ:

<html>
  <body>
    <h1>Hello</h1>
  </body>
</html>

---

2. CSS (Style)

* Tạo giao diện đẹp
* Flexbox, Grid (quan trọng)

Ví dụ:
.container {
display: flex;
justify-content: center;
}

---

3. JavaScript (Logic phía client)

* Xử lý click, DOM, API

Ví dụ:
document.querySelector("button").onclick = function() {
alert("Clicked!");
}

---

4. DOM & EVENT

* DOM: thao tác HTML bằng JS
* Event: click, submit

Ví dụ:
document.getElementById("btn").addEventListener("click", function() {
console.log("OK");
});

---

5. FETCH API (Gọi backend)
   Ví dụ:
   fetch("http://localhost:3000/api")
   .then(res => res.json())
   .then(data => console.log(data));

========================================
III. BACKEND (NODEJS + EXPRESS)
===============================

1. Cài đặt

* npm init -y
* npm install express

---

2. Tạo server cơ bản
   const express = require("express");
   const app = express();

app.get("/", (req, res) => {
res.send("Hello Backend");
});

app.listen(3000);

---

3. Middleware
   app.use(express.json());

---

4. API (RESTful)
   GET: lấy data
   POST: thêm data
   PUT: sửa data
   DELETE: xoá data

Ví dụ:
app.post("/user", (req, res) => {
res.send(req.body);
});

========================================
IV. DATABASE (MONGODB)
======================

1. Cài mongoose
   npm install mongoose

---

2. Kết nối DB
   const mongoose = require("mongoose");

mongoose.connect("mongodb://127.0.0.1:27017/test");

---

3. Tạo Schema
   const userSchema = new mongoose.Schema({
   name: String,
   age: Number
   });

const User = mongoose.model("User", userSchema);

---

4. CRUD

CREATE:
await User.create({ name: "A", age: 20 });

READ:
const users = await User.find();

UPDATE:
await User.updateOne({ name: "A" }, { age: 25 });

DELETE:
await User.deleteOne({ name: "A" });

========================================
V. KẾT NỐI FRONTEND + BACKEND
=============================

Frontend gọi API:
fetch("/user", {
method: "POST",
headers: { "Content-Type": "application/json" },
body: JSON.stringify({ name: "Test" })
});

Backend nhận:
app.post("/user", (req, res) => {
console.log(req.body);
});
